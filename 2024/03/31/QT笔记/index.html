窗体风格

QWidget 变量名（*父窗口, 窗体风格参数）;

setWindowModality(Qt::WindowModality windowModality)
指定了模态窗口的行为。(如：先用窗口3，关闭了窗口3才能使用其他)

参数 详情
Qt::NonModal 该窗口不是模态的，也不会阻塞其他窗口的输入。
Qt::WindowModal 该窗口是单一窗口层次结构的模态，并阻止输入到其父窗口、所有祖父窗口以及其父和祖父窗口的所有兄弟窗口。
Qt::ApplicationModal 该窗口对应用程序来说是模态的，并阻止所有窗口的输入。

QLineText 掩码字符

字符（必须输入） 字符（可留空） 含义
A a 只能输入 A-Z,a-z
N n 只能输入 A-Z,a-z,0-9
X x 可以输入任意字符
9 0 只能输入 0-9
D d 只能输入 1-9
# 只能输入+，-，0-9
H h 只能输入十六进制字符(0-9,a-f,A-F)
B b 只能输入二进制字符(0,1)
> > 后面的字母字符自动转换为大写
< < 后面的字母字符自动转换为小写
! ! 停止字母字符的大小写转换
\ \ 讲该表中的特殊字符正常显示作用分隔符
